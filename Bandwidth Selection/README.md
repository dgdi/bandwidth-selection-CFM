#Bandwidth Selection

This repo provides an R implementation of the algorithm proposed in : [Optimal bandwidth selection for conditional efficiency measures: A data-driven approach](http://www.sciencedirect.com/science/article/pii/S0377221709002148) by Luiza Badin, Cinzia Daraio , Léopold Simar.

###Descrption of files in the Repo

* Ker_LSCV_IN.R : the input version of Ker_LSCV_OUT.R
* Ker_LSCV_OUT.m : the MATLAB function provided in the paper
* Ker_LSCV_OUT.R : the proposed R function implementing Ker_LSCV_OUT
* README.md : this file
* Validation : this folder stores simulations to assess the reliability of the R function
